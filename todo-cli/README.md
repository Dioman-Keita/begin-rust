# todo-cli

`todo-cli` is a small Rust command-line application for managing a simple list of tasks.

## Features

- Add a new task
- List existing tasks
- Exit the program

## Requirements

- Rust toolchain installed via [rustup](https://rustup.rs/)

## Run

From the project directory:

```bash
cargo run
```

## Build

```bash
cargo build
```

## Notes

This project currently stores tasks only in memory, so the list is reset when the program exits.
